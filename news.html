New Version of TPA!

A new version has been released, Version 1.3! This new version allows you to control the flashing lights on the "Epilepsy Mode" feature. Increasing this will make the lights flash faster. The new version also adds smoother control to the character and updates the gravity control. Lastly, some smaller graphical and other random changes have been made to make it look nicer.



New Version of TPA!

A new version has been released, Version 1.2! This version adds a more random aspect to the block placement, adds flashing lights using the "Epilepsy Mode" feature (next update will contain a choice to turn them off). More work is going to be added onto it in the future as it is still a beta product.



Complete Rework of TPA!

The Pawn Adventure has been completely reworked into a much better version that it was before. The Platformer section has now become an infinite cycle. More work is going to be added to the future as this is still a beta product.
